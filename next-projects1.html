VOWELS & CONSONANTS

This program checks whether in a given string how many number of vowels are there
and how many number of consonants are there in the string.

PALINDROME

What this program does is whatever string you have entered it will check whether
it is a palindrome or not.
A palindrome is one when you reverse a string it should be the same as normal string
otherwise it is not a palindrome.


#palindrome using slicing

a=input('ENTER THE STRING')
b=a[::-1]
if a==b:
    print(a,' IS PALINDROME')
else:
    print(a,' IS NOT PALINDROME')
#palindrome using while loop
a=input('ENTER THE STRING')
b=0
c=-1
k=len(a)
count=1
w=(k+1)/2
while(count<=w):
    b=b+1
    c=c-1
    count=count+1
    if (a[b]!=a[c]):
        print(a,' IS NOT PALINDROME')
        break
if (a[b]==a[c]):
        print(a,' IS PALINDROME')
