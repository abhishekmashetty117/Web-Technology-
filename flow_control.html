Flow Control



if else statement 
Conditions tests if something is True or False, and it uses Boolean values
to check that.
You see that conditions are either True or False (with no quotes!).


Syntax 
{If ...else}
	if condition:
		statements

	else:
		statements


nested if statement 
You place an if statement inside another statement.This process is called
nesting and enables you to make complex decisions based on different
inputs.


Syntax 
if condition:
	if condition:
		statements
	else:
		statements
else:
      statements


if..elif..else statements 
The elif statement is simply a contradiction of the else if statement.

Syntax 
If ...else

    if condition:
        statements

    elif condition:
 statements

    else:
 statements


for loop 
for loops are traditionally used when you have a block of code which you
want to repeat a fixed number of times.The Python for statement iterates over
the members of a sequence in order, executing the block each time

Syntax 
for iterating_var in sequence:
   statements(s)


while loop 
The while loop in Python is used to iterate over a block of code as long as
the test expression (condition) is true.We generally use this loop when we don't
know beforehand, the number of times to iterate.

Syntax 
while (expression) :
    statement_1
    statement_2
    ......
else :
    statement_3
    statement_4
    ......

break statement 
break may only occur syntactically nested in a for or while loop, but not
nested in a function or class definition within that loop.It terminates the
nearest enclosing loop, skipping the optional else clause if the loop has one.

If a for loop is terminated by break, the loop control target keeps its current
value. When break passes control out of a try statement with a finally clause,
that finally clause is executed before really leaving the loop.

Syntax 
break_stmt:     "break"


continue statement 
continue may only occur syntactically nested in a for or while loop, but
not nested in a function or class definition or try statement within that loop.
It continues with the next cycle of the nearest enclosing loop.

Syntax 
continue_stmt:  "continue"


switch case expression 
A switch-case statement lets you control the flow of the program based on
the value of a variable or expression. In particular, if the variable or
expression that youâ€™re testing has a number of different of possible values,
you could execute a block of code for each separate value

Syntax 
switch EXPR:
    case EXPR:
        statement
        break
    case EXPR:
        statement
        break
    ...
    case default:
       statement
       break
pass statements 
pass is a null operation -- when it is executed, nothing happens.
It is useful as a placeholder when a statement is required syntactically,but no
code needs to be executed, for example:

Syntax 
pass_stmt:      "pass"
def f(arg): pass    # a function that does nothing (yet)
class C: pass       # a class with no methods (yet)


The infinite loop 
A loop becomes infinite loop if a condition never becomes FALSE. You must
use caution when using while loops because of the possibility that this condition
never resolves to a FALSE value. This results in a loop that never ends. Such a
loop is called an infinite loop.
An infinite loop might be useful in client/server programming where the server
needs to run continuously so that client programs can communicate with it as and
when required.
